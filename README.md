# 👓 Lensshine Software

A full-stack optical store management system built for managing customers, orders, billing, invoices, and payment tracking for Lensshine Optical Store.

---

## Features

- ✅ Customer Order Management
- ✅ Optical Prescription Entry
- ✅ Payment Tracking System
- ✅ Dashboard Analytics
- ✅ Customer History Search
- ✅ Invoice Generation
- ✅ Email Invoice Sending
- ✅ MongoDB Cloud Database Integration
- ✅ Responsive UI
- ✅ Live Deployment

---

## Tech Stack

### Frontend
- React.js
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js
- Spring Boot
- MongoDB Atlas
- Mongoose
- Nodemailer

### Deployment
- Netlify (Frontend)
- Render (Backend)
